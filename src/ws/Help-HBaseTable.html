hbase table

The hbase table panel allows searching connected HBase databases.

The top part of the panel sends requests to the HBase. It searches for data following several paterns. Firts three criteria are exclusive.

  • Exact key: Searching for just one row specifying its exact key. Very fast search.
  • Prefix key: Searching for rows with keys matching specified string as a prefix.
  • Search columns: Searching for rows with several columns matching specified strings as substrings. The request should have format family:column:value,......
  • Show columns: Get only specified columns. This field is compatible with any other criterium. The request should have format family:column,......
  • Limit
    • limits number of rows to be demanded.
  • From, Till
    • limits search to a specific time interval (of data entries into database).

      The bottom pannel manipulates already received data, results of the HBase search. It allows to perform further selections, sorting according to any column, visualising binary data (e.g. FITS files) using appropriate plugins, drawing time evolution of selected columns,...
