hbase table

The hbase table panel shows and manipulates already HBase data, results of the HBase search. It allows to perform further selections, sorting according to any column, visualising binary data (e.g. FITS files) using appropriate plugins, drawing time evolution of selected columns,...

Each alert entry can be shown in the Graph View. Its binary entries can be visualised using appropriate plugins (for example JS9 plugin for FITS images) or in external applications.

Numerical variables can be selected for generation of several plots. The Evolution Plot shows time evolution of selected variables (in any selection column). The Scatter Plot shows a 2D plot of selected variables; the first and second column give x and y coordinates (only the first selection is taken), the third column gives the cell value (may be absend or several variables van be shown simultaniously), the fourth column gives the cell value which be be used to group results (by color etc.).
