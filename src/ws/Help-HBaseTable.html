hbase table

The hbase table panel shows and manipulates already HBase data, results of the HBase search. It allows to perform further selections, sorting according to any column, visualising binary data (e.g. FITS files) using appropriate plugins, drawing time evolution of selected columns,...

Each alert entry can be shown in the Graph View. Its binary entries can be visualised using appropriate plugins (for example JS9 plugin for FITS images) or in external applications.

Numerical variables can be selected for generation of several plots. The Evolution Plot shows time evolution of variables selected in x and y columns). The Scatter Plot shows a 2D (x*y) plot of selected variables. The z column gives the cell value (may be absent). The c column gives the cell value which be be used to group results (by color etc.); if it is not selected, the groups will be created to distinguish different z columns.
